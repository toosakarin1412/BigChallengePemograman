### Cara melakukan compile

1. Menggunakan <b>makefile</b>
> `make wordscore`

2. Jika tidak bisa menggunakan <b>makefile</b> maka gunakan perintah dibawah ini
> `gcc -o wordScore wordScore.c imple.c -Wall`

### Cara menjalankan program
1. Pada Windows
> `./wordscore.exe`

2. Pada Linux
> `./wordscore`
